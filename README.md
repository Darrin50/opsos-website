# OpsOS Website

Marketing website for OpsOS — Warehouse Operations Intelligence Platform.

## Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Deployment:** Vercel

## Pages

- `/` — Homepage with hero, modules, pain points, FAQ, demo form
- `/features` — All 6 modules explained in detail (OpsPulse, FlowAI, WasteWatch, ShiftAdvisor, Ask OpsOS, SafetyShield)
- `/who-its-for` — Industry pages (Tier 2–3 Automotive, Steel, Fabricators, Distribution, Grocery)
- `/pricing` — Free Trial / Growth ($497/mo) / Enterprise
- `/blog` — Blog index
- `/blog/[slug]` — Individual blog posts with Article + FAQ JSON-LD

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

1. Connect this repo to Vercel at [vercel.com/new](https://vercel.com/new)
2. Select the `opsos-website` repo
3. Framework: Next.js (auto-detected)
4. No environment variables required for basic deployment
5. Deploy — Vercel handles the rest
